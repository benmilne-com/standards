# CSF - vendored reference

A pinned copy of the Commons Stablecoin Format, kept here as part of the benmilne.com standards
reference so the set is self-contained and version-stable.

- **Format**: Commons Stablecoin Format (CSF) `v1.4.7`
- **Author**: Brale.xyz
- **License**: MIT, (c) 2025 Brale (see [LICENSE.brale](LICENSE.brale))
- **Source**: https://github.com/Brale-xyz/commons
- **Vendored at commit**: `3e696a6ee4a5d59c8e3a48d4a593dbf5d3a3d8e2`

Upstream remains canonical. See [README.upstream.md](README.upstream.md) for the original Brale
README. We track a pinned copy here so the standards reference does not depend on upstream
availability or drift; update it deliberately when CSF advances.

CSF is the sibling of GSF ([../gsf](../gsf)): CSF documents how value moves step by step (sequence
diagrams), GSF documents how entities are structurally related (graphs). Same three value-layer
primitives, two modalities.
